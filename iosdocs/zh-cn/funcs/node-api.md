## 说明
- 节点模块函数主要是跟节点操作相关联
- 该模块没有模块前缀，可以直接调用函数
- 由于IOS的限制，可能部分机器获取节点过慢，建议获取节点后进行锁定节点后，在进行查找
- 也可以使用设置节点获取参数的方式进行



## 获取节点函数

## getNodeInfo 获取节点集合

* label属性全匹配
* @return 字符串

> ```javascript
> function main(){
> var data = label("aaa").;
> logd(data);
> }
> main();
> ```
