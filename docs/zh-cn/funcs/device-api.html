1. 说明

  • 设备模块函数主要是跟设备信息相关联
  • 设备模块的对象前缀是device，例如 device.getIMEI()这样调用

2. 函数

2.1. device.getIMEI()

  • 获取手机的IMEI
  • @return 字符串

function main(){
    var imei = device.getIMEI();
    toast(imei);
}
main();

2.2. device.getScreenWidth()

  • 取得屏幕宽度
  • @return 整型

function main(){
    var width = device.getScreenWidth();
    toast(width);
}
main();

2.3. device.getScreenHeight()

  • 取得屏幕高度
  • @return 整型

function main(){
    var height = device.getScreenHeight();
    toast(height);
}
main();
