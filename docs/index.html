1. EasyClick单机版

解放双手，可编程式、智能化自动点击App；只要会敲键盘就能进行编程！

  • 简单易编程
  • 丰富的API，图像识别
  • 可脱机运行，可单独发布
  • 支持无障碍/免ROOT运行
  • 智能IDE编程支持
  • ...

2. EasyClick企业版

云控平台，数千台手机同步运行，节省企业测试成本

  • 不局限地域，任意地方部署设备
  • 云端画面实时同步
  • 定时/非定时执行任务
  • 人机控制同步，人工可介入控制替代自动操作
  • 自带数据分析功能，实时数据报表
  • ...

3. EasyClick能做什么

  • App爬虫等
  • 残障人士的辅助，例如语音读书等
  • 自动化测试App
  • ...

4. 适合人群

  • 有意学习自动化测试脚本的人员
  • 希望用自动操作代替手动的人员
  • 企业App测试团队及测试人员
